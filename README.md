# Crazy Parser

A light-weight parser combinator.

## Install

```sh
npm i crazy-parser
```

## Development

### Using make

Please make sure you have **node**, **tsc** and **vitest** installed.

#### Build

```sh
make
```

#### Build a tarball

```sh
make pack
```

#### Clean up

```sh
make clean
```

#### Test

```sh
make test
```

### Using npm

Please make sure you have **tsc** and **vitest** installed.

#### Build

```sh
npm run build
```

#### Build a tarball

```sh
npm run build
npm pack
```

#### Clean up

```sh
npm run clean
```

#### Test

```sh
npm run clean
npm run test
```
